When you download an app from the internet, macOS adds a "quarantine" attribute to it. This tells Gatekeeper to verify that the app is signed by a registered developer.

Since this app is not signed (code signing costs $99/year), Gatekeeper blocks it. The script simply removes the quarantine attribute, telling macOS to trust the app.
